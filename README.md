# Lycka Network Househelps Bureau — Website

A professional 4-page static website for Lycka Network Househelps Bureau, built with HTML, CSS, and vanilla JavaScript.

## 📁 Project Structure

```
lycka-network/
├── index.html              # Homepage
├── css/
│   ├── main.css            # Global styles (nav, hero, footer, utilities)
│   └── packages.css        # Packages page specific styles
├── js/
│   └── main.js             # Scroll animations, mobile nav, reveal effects
├── pages/
│   ├── packages.html       # All 6 packages + add-ons
│   ├── services.html       # Service types + placement process
│   ├── about.html          # Mission, vision, team, stats
│   └── contact.html        # Contact form + social links
└── README.md
```

## 🎨 Design

- **Colors:** Deep Purple `#3B1A6B`, Gold `#C9A227`, Off-white `#FAF8F5`
- **Fonts:** Playfair Display (headings) + DM Sans (body)
- **Style:** Professional, luxury domestic agency aesthetic
- **Responsive:** Mobile-first, works on all screen sizes

## 📄 Pages

| Page | Description |
|------|-------------|
| `index.html` | Hero, why choose us, packages preview, coverage, CTA |
| `packages.html` | All 6 packages with details + add-on services |
| `services.html` | 6 service types + 4-step placement process |
| `about.html` | Story, mission/vision, stats, team |
| `contact.html` | Contact form, methods, social media links |

## 🚀 Getting Started

Simply open `index.html` in any browser — no build tools required.

To host online, upload the entire folder to:
- GitHub Pages
- Netlify (drag & drop the folder)
- Any standard web host
